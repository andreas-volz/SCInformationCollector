Conditions

Conditions

Sets of 20 "0"'s - if no condition - Always 16 sets

20 Bytes

- Unknown (Long)
- String # for location (Long)
- Player who condition affects (Long)

IF NO SWITCHES or RESOURCES needed in this trigger:
- Number of units affected (long)
ELSE:
- 3 Unused (Bytes)
- Switch value: 2 = Set, 3 = Cleared; OR Resource Type: 0 = Ore, 1 = Gas, 2 = Ore And Gas (Byte)

- Unit Type (Integer) - See Unit List for 0-228, The rest as follows:

  • 229 = Any Unit

  • 230 = Men

  • 231 = Buildings

  • 232 = Factories


- Qualified #: 0 = Or more, 1-255 = Or less (Byte)
- Condition Type (Byte) - See Conditions List
- Switch # / Score # to use (Byte)
- Switch State: 0 = enabled, 2 = disabled (Byte)
